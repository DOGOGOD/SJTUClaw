# Material Summary Skill

## 能力范围

此 skill 用于汇总和整理学习资料，覆盖以下场景：
- 整理多份课堂笔记为一个知识体系
- 汇总项目文档和调研资料
- 提取多个文件的核心知识点
- 生成复习大纲或知识点清单

## 工作流程

1. **确定范围**：从用户任务中确认：
   - 要汇总的材料范围（具体文件或目录，在 workspace 内）
   - 汇总目标（知识点总结 / 项目概览 / 复习提纲）
   - 保存路径

2. **扫描材料**：
   - 使用 list_dir 列出相关目录
   - 使用 read_file 逐一读取材料内容
   - 对大文件可以只读取关键部分（标题、摘要、结论）

3. **提取和归类**：
   - 识别每份材料的核心主题
   - 按主题归类，合并重复内容
   - 标注信息来源（来自哪个文件）

4. **生成摘要**：
   - 结构：

   ```
   # 学习材料汇总

   ## 概述
   （简要说明汇总范围和材料数量）

   ## 主题一：XXX
   - 核心知识点
   - 来源文件
   - 重要细节

   ## 主题二：XXX
   ...

   ## 关键概念索引
   （按字母/拼音排列的关键术语和定义）

   ## 后续学习建议
   （基于现有材料的缺口，建议补充的方向）
   ```

5. **保存**：使用 overwrite_file 写入 workspace，走 approval 流程

## 注意事项

- 保持客观，不添加材料中没有的观点
- 标注每一条信息的来源文件
- 对不确定的内容标注"[待确认]"
