# Presentation Outline Skill

## 能力范围

此 skill 用于生成课堂展示 / 演讲的大纲，覆盖以下场景：
- 课堂主题展示 / 小组汇报
- 学术演讲 / 论文答辩
- 项目汇报 / 工作汇报
- PPT 页面结构设计

## 工作流程

1. **理解需求**：
   - 展示主题
   - 时间限制（如 5 分钟 / 15 分钟）
   - 受众背景（同学 / 老师 / 专业人士）
   - 参考材料（workspace 中的文件）
   - 保存路径

2. **读取参考材料**（如有）：
   - 使用 read_file 读取 workspace 中的相关文件
   - 提取关键论点、数据和案例

3. **设计大纲结构**：
   - 标准展示结构：

   ```
   # 展示大纲：[主题]

   ## 基本信息
   - 主题：XXX
   - 时长：X 分钟（约 X 张幻灯片）
   - 受众：XXX

   ## 幻灯片 1：封面
   - 标题、姓名、日期

   ## 幻灯片 2：目录 / 议程
   - 本次展示的主要内容概览

   ## 幻灯片 3-N：正文
   ### 幻灯片 X：[小标题]
   - 核心要点 1
   - 核心要点 2
   - 支撑数据 / 案例
   - 过渡语 → 下一页

   ## 幻灯片 N+1：总结
   - 回顾核心观点
   - 结论

   ## 幻灯片 N+2：Q&A
   - 致谢
   - 欢迎提问

   ## 演讲备注
   - 每页的关键演讲词
   - 时间分配建议
   ```

4. **保存大纲**：
   - 使用 overwrite_file 写入，走 approval

## 设计原则

- 每张幻灯片不超过 3-5 个要点
- 要点用简洁的短语，不是完整句子
- 过渡语帮助听众跟上思路
- 时间分配要合理（开场 10%，正文 70%，总结 15%，Q&A 5%）
